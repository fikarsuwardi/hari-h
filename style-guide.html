Tone aturan

Owner App: brand hijau (--brand) di tombol utama, ikon, dan brand mark. Card putih di atas paper cream. Status pakai semantic colors (paid/upcoming/unpaid/overdue).

Admin Panel: ditandai class .admin-theme. Brand mark + nav active state ganti ke --deposit (#3f5e6c). Tombol primary di form admin pakai background --deposit. Memberi sinyal visual jelas: "ini area internal".

Display vs Sans: Fraunces hanya untuk display & angka penting (Rupiah, judul). Sans Jakarta untuk semua body, label, tombol, dan UI controls. Tidak campur dalam satu kalimat.
